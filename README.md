# PickleCity Tournament Manager V4.10.18 KO Player Names

## Đã sửa
- Khi Tứ kết kết thúc, đội thắng đi tiếp lên Bán kết kèm đầy đủ tên VĐV.
- Khi Bán kết kết thúc, đội thắng đi tiếp lên Chung kết kèm đầy đủ tên VĐV.
- Tranh giải 3 cũng nhận đúng đội thua Bán kết và tên VĐV.
- Màn hình Nhập điểm và Nhánh đấu đều ưu tiên hiển thị tên đội + tên VĐV ở các vòng Knockout.

## Cách update bằng GitHub Desktop
1. Giải nén file ZIP này.
2. Copy toàn bộ nội dung vào thư mục project GitHub hiện tại.
3. Mở GitHub Desktop.
4. Commit: `V4.10.18 - KO Player Names`
5. Push origin.
6. Cloudflare Pages tự deploy.
