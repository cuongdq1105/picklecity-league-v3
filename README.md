# PickleCity Tournament Manager V4.10.10 Standings Bracket Names

Bản cập nhật tên đội/VĐV ở BXH và nhánh đấu.

## Đã sửa
- BXH hiển thị rõ **Tên đội / VĐV** trong cùng một cột.
- BXH công khai cũng hiển thị tên đội + tên VĐV.
- Nhánh đấu lấy đúng:
  - A1/B1/C1/A2/B2/C2/Best3
  - Tên đội thật
  - Tên VĐV
- File in nhánh đấu cũng ưu tiên hiển thị tên đội/VĐV.

## Lưu ý sử dụng
Sau khi nhập xong điểm vòng bảng:
1. Vào BXH kiểm tra nhất/nhì/ba.
2. Bấm Sinh/Tạo lại nhánh.
3. Nhánh đấu sẽ điền đúng tên đội và VĐV.
